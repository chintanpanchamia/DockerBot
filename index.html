Setup an Environment?

Mention the bot on a channel, or open a private chat with the bot, and then use the commands:

  • Set this up for me _link_
  • Can you set this environment up? _link_
  • _link_ (Or just the link itself)

Get a popular technology stack?

Mention the bot on a channel, or open a private chat with the bot, and then use this command:

  • Get me the _name_ stack
EnvBot currently can provide with Docker images for the following technologies/stacks:
  • LAMP
  • MEAN
  • Python
  • Java
  • Apache Spark
