DockerBot

How to ...

Setup an Environment?

Mention the bot on a channel, or open a private chat with the bot, and then use this command:

  • docker file _link_

Get a Docker Image for a repository of your choice?

Mention the bot on a channel, or open a private chat with the bot, and then use this command:

  • docker image _link_

Get a Docker Image of a popular technology stack?

Mention the bot on a channel, or open a private chat with the bot, and then use this command:

  • request docker image _name_
DockerBot currently can provide with Docker images for the following technologies/stacks:
  • LAMP
  • MEAN
  • Python
  • Redis
  • Postgres
  • Ubuntu
  • Mysql
